# Product Overview

## Purpose

Task Manager - Um aplicativo web moderno para gerenciamento de tarefas pessoais, desenvolvido com foco em usabilidade, acessibilidade e interface responsiva.

## Key Features

- **CRUD Completo**: Criar, visualizar, editar e excluir tarefas
- **Persistência Automática**: Dados salvos no LocalStorage do navegador
- **Interface Responsiva**: Design mobile-first (320px+) com suporte a desktop (768px+)
- **Acessibilidade WCAG 2.2 AA**: Navegação por teclado, leitores de tela, contraste adequado
- **Notificações**: Feedback visual para todas as ações do usuário
- **Validação**: Validação de formulários com mensagens de erro claras
- **Tratamento de Erros**: Error boundaries e recuperação de falhas

## Technology Stack

- React 19 + TypeScript
- Vite (build tool)
- CSS3 (sem frameworks)
- LocalStorage API
- Vitest + React Testing Library
- Property-based testing com fast-check

## Development Methodology

- **Spec-Driven Development**: Requirements → Design → Tasks → Implementation
- **Test Coverage**: 90% minimum (line, branch, function)
- **Accessibility First**: WCAG 2.2 AA compliance
- **Conventional Commits**: Structured commit messages
- **AI-Assisted Development**: Built with Kiro AI Assistant
